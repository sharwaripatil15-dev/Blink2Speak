# Blink2Speak
Blink2Speak is an assistive communication system that helps people with severe speech or mobility impairments communicate using eye blinks. A camera or sensor detects intentional blinks and converts them into text or speech. It is useful for patients with conditions like ALS, allowing them to express needs and interact with caregivers and family.
The system works by using a camera or eye-detection sensor to continuously monitor the user’s eye movements. When the user intentionally blinks or follows a specific blinking pattern, the system detects this input and processes it through a programmed interface. These blink signals can be used to select letters, words, or predefined messages displayed on a screen. Once a message is selected, the system can either display the text or convert it into voice output using a text-to-speech module. This allows the user to communicate basic needs such as asking for help, requesting water, or expressing discomfort.
The project focuses on creating an affordable, user-friendly, and reliable assistive technology that can be easily used in hospitals, rehabilitation centers, or at home. With the help of computer vision, sensors, and basic software algorithms, Blink2Speak demonstrates how technology can improve the quality of life for people with disabilities. Overall, this system promotes independence, dignity, and better interaction for individuals who would otherwise struggle to communicate with the world around them.
